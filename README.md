## Overview

Aplicacao web React para controlar TVs LG webOS via Wi-Fi. Permite controle remoto completo da TV atraves do navegador.

---

## Features

| Feature | Descricao |
|---------|-----------|
| **Controle Remoto** | Interface de controle completa para TVs LG |
| **Wi-Fi** | Conexao via rede local sem fio |
| **webOS** | Compativel com TVs LG webOS |
| **Responsivo** | Funciona em desktop e dispositivos moveis |
